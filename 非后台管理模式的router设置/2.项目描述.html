1.配置好了antd，实现按需加载antd的组件
2.开启css modules模式 1> 如果使用sass 需要安装node-sass ,然后文件命名需要设置成 .module.sass或者.module.scss文件 2.如果使用less 需要在getStyleLoaders 方法里面设置 module:true 然后命名文件为.module.less
3.less文件引入less文件，最后面一定要加上；否则报错 @import './reset.less'；
4.nginx 的配置 https://cloud.tencent.com/developer/article/1198752 在nginx.conf下面配置 https://cloud.tencent.com/developer/article/1183138 分开配置 server { listen 80; server_name cloud.fangyuanxiaozhan.com; location / { proxy_set_header X-Real-IP $remote_addr; proxy_set_header Host $http_host; proxy_pass http://0.0.0.0:8080; } } //实现了fangyuanxiaozhan.com:8080映射到 cloud.fangyuanxiaozhan.com 错误部分在 var/log/nginx 里面，可以查看报错信息
